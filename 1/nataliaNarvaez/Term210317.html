LA TERMINAL

Programa presente en todos los Sistemas Operativos para darle órdenes al sistema. Dentro de estas órdenes está crear carpetas o archivos, colocando la extensión, mover un archivo de un lugar a otro y mucho más.

COMANDOS EN LA TERMINAL

ls: lista de archivos

ls -l: Me muestra carpetas

ls -a: archivos ejecutables

pwd: donde estoy

cd: abre la carpeta a donde deseamos ir

clear: limpia la pantalla

nano: comando que hace algo que no entiendo bien

rm: nombreArchivo.extensión se elimina

Para guardar: dentro de nano para guardar: ctrl + S

se pone el nombre y se guarda con enter y X para ir atrás

mv: comando para mover y renombrar

Script: Serie de comandos escritos de forma secuencial que se ejecutan cuando uno los llama. Son una herramienta muy utilizada en informática para hacer backups

Generar un script.bat

.bat para hacer un script en microsoft

Comando de partición de disco:

df: Muestra el consumo de las cosas en la memoria

df -h: Comando para humanos que muestra la misma información pero más bonita

Comando para comprimir archivos:

tar -zcvf respaldo.tar.gz texto11.txt
